## 신규 기능

### 1. 공고 맞춤 면접 모드
채용 공고를 기반으로 맞춤형 면접 질문을 생성하는 새로운 모드.

- **공고 입력 방식**: 회사명/직무(선택) + 자격요건/우대사항 스크린샷 Ctrl+V 붙여넣기
- **질문 구성**: 총 5개 (공고 기반 2~3개 + 인성/기술 2~3개, 공고 내용 양에 따라 유연 조절)
- **질문 톤**: "저희 팀에서는..." 등 해당 회사 면접관 시점으로 생성
- **비전 모델**: GPT-4o-mini로 캡처 이미지 내 텍스트 인식 후 질문 생성
- **기업 규모**: 선택 가능 / 면접 유형, 질문 수: 자동 설정 (behavioral, 5개 고정)

### 2. 하드모드
준비 시간 없이 즉시 답변해야 하는 실전 모의면접 모드.

- **타이핑 애니메이션**: 질문이 55ms/자 속도로 타이핑되며 표시
- **3초 카운트다운**: 타이핑 완료 후 캠 화면 중앙에 3→2→1 대형 숫자 표시
- **자동 녹화 시작**: 카운트다운 종료 후 버튼 없이 자동 녹화
- **답변 제한시간**: 인성 질문 3분, 기술 질문 5분
- **타이머 UI**: 질문 카드 하단에 남은 시간 표시, 30초 이하 시 빨간색 경고
- **시간 초과**: 자동 답변 종료 → 다음 질문으로 이동
- 면접 유형/질문 수는 일반 모드와 동일하게 선택 가능

### 3. SetupPage 레이아웃 재구성
좌측 모드 선택 + 구분선 + 우측 콘텐츠 레이아웃으로 전면 재설계.

- **좌측 네비게이션**: 일반 모의면접 → 하드모드 → 공고 맞춤 면접 (텍스트 + 좌측 인디케이터 바)
- **레이아웃 안정성**: 모드 전환 시 좌측 네비/구분선/기업 규모 위치 불변
- **기업 규모 상단 고정**: 양쪽 모드에서 동일 위치
- Windows/Mac 캡처 단축키 안내 포함

## 리포트 뷰 개선

### 질문 중심 재구성
기존 면접관 아코디언 내 질문 중첩 구조 → 질문 단위 통합 뷰로 변경.

- **면접관별 평가**: 점수 + 총평 + 강점/개선점 카드 (간결)
- **질문별 평가**: 질문 풀텍스트 + 면접관 평균점수 토글
  - 펼치면: 답변 영상, 면접관별 점수/코멘트(바 그래프), 비언어 분석, 모범답안
- 질문 텍스트 잘림 문제 해결 (풀텍스트 표시)

## STT 품질 개선

### 텍스트 잘림 현상 해결
답변이 뒤에서 잘리는 문제의 원인을 분석하고 수정.

- **반복 제거 완화**: 유사도 기준 80% → 95% (합법적 답변 보존)
- **환각 정규식**: 동일 구절 3회+ 연속 반복만 제거
- **max_tokens 명시**: 교정 8192, 평가 16384 (응답 잘림 방지)

## 문서 질문 개선

- **질문 수 고정**: 일반 4문항→1개, 5문항→2개 문서 질문 (총 수 유지, 추가가 아닌 대체)
- **PDF 처리**: pdfjs-dist로 교체 + 이미지 PDF OCR 대응
- **프롬프트 강화**: 문서 내용 직접 인용 규칙 강화
